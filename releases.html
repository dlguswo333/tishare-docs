0.2.08/1/2021
tiShare Desktop v0.2.0 is out. tiShare Desktop v0.2.0 is tested in accordance with tiShare Mobile v0.2.0. It is advised to match app versions. tiShare plans to have integrated versioning in both Desktop and Mobile. v0.2.0 contains the following bug fixes and improvements:
  • Add a new icon.
  • Improve local network detection mechanisms.
  • When folders are added to item list, every subitems existing at the time of clicking Send will be transferred.
  • Show opponent ID in top right of JobView.
  • Delete JobView automatically if ended by on this side, not by the opponent.
  • Server and Client have integrated indexing mechanisms.
  • Improve data transfer stabilities.
  • Fix a bug where even if the app is open to the network, it cannot be searched due to wrong bind address in Linux.
  • Fix a bug where uncompletely received files are not deleted automatically.
  • Fix miscellaneous bugs.
